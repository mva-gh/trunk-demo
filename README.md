# trunk_demo

![](./doc/img/logo.jpeg)

Demo shows how to setup Rust + WASM project using [trunk](https://github.com/thedodd/trunk/blob/master/Trunk.toml) as build tool.

## How to start

- Clone repo
- Install trunk: `cargo install trunk`
- Build and run: `trunk serve`
